# Task Definition

Given a research question, the agent must:

1. Gather relevant information from available documents
2. Extract key facts
3. Track sources for each fact
4. Identify missing information
5. Produce a structured summary with citations

## Constraints

* The agent must decide when to search more
* The agent must decide when it has enough information
* The agent must not hallucinate sources
* The process must be step-by-step and inspectable

## Success Criteria

* Clear, structured output
* Evidence-backed statements
* Transparent reasoning steps
